Integrator Manual
====================

.. _create-an-element:

Creating a H5P content element
==============================

* Select a storage folder from the page tree
* Click the H5P icon
* Click the 'add' icon

.. figure:: ../Images/Integrator/New.png
   :class: with-shadow
   :width: 500px
   :alt: New H5P Element

* After creating and saving a new element you will see a preview

.. figure:: ../Images/Integrator/Preview.png
   :class: with-shadow
   :width: 500px
   :alt: Preview

* Listing the content elements in the folder

.. figure:: ../Images/Integrator/Content_on_page.png
   :class: with-shadow
   :width: 500px
   :alt: Content in the selected folder

Displaying the H5P content in TYPO3
===================================

To display the created H5P element on the site you will need to create a TYPO3 content element of type H5P

.. figure:: ../Images/Integrator/Content.png
   :class: with-shadow
   :width: 500px
   :alt: TYPO3 H5P content elements

The `view` element will render the H5P content

.. figure:: ../Images/Integrator/View.png
   :class: with-shadow
   :width: 500px
   :alt: TYPO3 H5P view element
